# 실제 스크린샷 넣는 곳

기본 화면은 앱 제목·아이콘으로 된 **스타일 썸네일**로 나옵니다.
학생이 직접 앱을 실행해 찍은 실제 화면으로 바꾸고 싶으면,
이 폴더에 **`앱제목.png`** 파일을 넣으세요. 자동으로 교체됩니다.

- 파일 이름 = 시트의 앱제목과 똑같이 (예: `단어장.png`, `링고 퀘스트.png`)
- 지원 형식: `.png`, `.jpg`, `.jpeg`, `.webp`
- 권장 비율: 가로:세로 = 4:3 (예: 800×600)
- 제목이 없는 앱(이도훈)은 `실시간 자리 배치 시스템.png`

파일을 넣지 않으면 스타일 썸네일이 그대로 유지되므로,
있는 것만 넣어도 됩니다.
